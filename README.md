# Quest through Cairo

A text-based adventure game set on Cairo Road in Lusaka, Zambia.

## Story

A massive pile-up has occurred on Cairo Road and a zombie apocalypse just broke out. You start at the Post Office and must fight your way through the undead to escape.

**Your quest (9/30):** Defeat the zombies and escape Cairo Road.

## How to Play

```bash
node src/adventureGame.js
```

## Requirements

- Node.js
